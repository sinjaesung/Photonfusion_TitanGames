## Overview
플레이어별 랜덤한 서로 다른 캐릭터로 접속 가능하며
각 캐릭터는 서로 다른 근접 공격 스타일을 가지고 있으며
맵에 존재하는 Network Monster AI와 전투할 수 있는
PVP / PVE 멀티플레이 게임입니다.

## Tech Stack
- Unity
- C#
- Photon Fusion
- NavMesh AI

## Key Features
- 실시간 플레이어 동기화
- Server Authoritative 구조
- 몬스터 AI 및 전투 시스템
- 네트워크 상태 동기화
